# AC 學期三作業 AlphaShop

## 表單實作

目前進度:Main、Header、Footer 已完成，dark-mode 已完成。

### 安裝

- 打開終端機 輸入指令 `git clone  'https://github.com/zebrrrra/S3_ALPHA-Shop.git'`
- 進入專案資料夾 `cd S3_ALPHA-Shop`
- 安裝所需套件 `npm install`
- 啟動專案 `npm start`
- 在瀏覽器輸入 http://localhost:3000 可瀏覽本專案

### 環境設置

- React 18.2.0
- Node.js 14.16.0
- npm 6.14.11
